蛋城争霸白皮书草稿

战局胜负规则

a. 每个玩家都可以创建战局，可以指定战局败方赔蛋率(10%-60%)，以及 助威赏蛋率(0.1%-%3)

b. 进行中的战局数量最多只能有 10 个

c. 连续10分钟，落后方没有追加投入，战局结束(最少追加双方差额10%)

d. 领先方不能再追加投入

e. 投入蛋蛋多的一方赢得战局的胜利

f. 战局双方阵营收回自己投入的全部蛋蛋的100%(冻结三天后返还到玩家账户)，剩余0%作为官方利润

g. 败方阵营交付自己收回的蛋蛋*败方赔蛋率，交给胜利阵营，作为战局赔蛋

h. 助威赏蛋 = 败方交付的战局赔蛋 * 助威赏蛋率

i. 胜利阵营的玩家，按照投入战局的蛋蛋的比例，分配失败方交付的 战局赔蛋 - 助威赏蛋
我获得的赔蛋 = 我投入的蛋蛋 / 胜利阵营投入的总蛋蛋 * (战局赔蛋 - 助威赏蛋)

j. 我获得的赔蛋，将冻结三天后，解冻到玩家账户中

k. 给胜利阵营助威的玩家，按照助威时间，分配助威赏蛋:
我获得的助威赏蛋 = 胜方打赏总蛋 / 胜方助威人数 *
( (胜方助威用户平均助威时间 - 战局开始时间) / (我的助威时间 - 战局开始时间) )

l. 如平局，则双方各自收回己方蛋蛋的100%(冻结三天后返回账户)

蛋蛋发行规则

a. 蛋蛋总量上限1.2亿，公开发行1亿，开发运营团队自留1000万，内购方式出售1000万

b. 开发运营团队自留的1000万，分四年解禁，开服满一年解禁25%，之后每半年解禁12.5%

c. 仅预注册玩家有资格参与内购，每人最高10万，内购截止时间3月7号22点，内购活动结束时立即销毁未售出的内购蛋蛋。

d. 第一年发行5000万，以后逐年衰减一倍，第一次发行时间为3月8号22点。

e. 3月8号第一天发行的分配规则为：具有内购资格的注册用户，平均分配开服第一天的发行量。

f. 3月9号开始之后的分配规则为：玩家的分配到的蛋蛋 = (玩家在当日战局中获得的蛋蛋+玩家在当日战局中失去的蛋蛋)
/ (所有玩家在当日战局中获得的蛋蛋+所有玩家在当日战局中失去的蛋蛋) * 当日发行总量

g. 当日战局指的是昨天22:00至今天22:00之间结束的战局，如两个时间之间没有结束任何战局，则当日的发行量全部销毁
